# Apache Jena Fuseki

## Build and run

```bash
DOCKER_BUILDKIT=1 docker build --build-arg JENA_VERSION=4.3.1 --build-arg JVM_ARGS="-Xmx8G" -t fuseki .

# Temporary container
docker run --rm \
  -p 3030:3030 \
  --name=fuseki fuseki

# Persistent container
docker run -d \
  -p 3030:3030 \
  --name=fuseki fuseki
```
